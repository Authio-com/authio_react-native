# Changelog

All notable changes to `@useauthio/react-native` are documented here. This
project adheres to [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] — 2026-06-12

### Changed
- **Renamed npm package `@authio/react-native` → `@useauthio/react-native`.** The
  original `@authio` scope could not be claimed on npm, so every Authio
  SDK now publishes under the organization scope `@useauthio`. Install
  with `npm install @useauthio/react-native` and update imports accordingly.
  The old `@authio/react-native` name is retired; releases below this entry were
  published (or prepared) under the old name and are kept for history.
